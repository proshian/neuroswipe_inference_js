Given a keyboard in json format like below create a javascript app with a keyboard drown on screen. The only thing needed from the app - logging coridnates of a swipe gesture

The keyboard json has three properties
 * width - keyboard width
* height - keyboard height
* keys - list of keys

Each key is a dict that has
* label or action - what should be written on the key
* x hibox - dict with x, y, w, h values: coordinates of left-top corner and width and height of the key

Each time user swipes on the keyboard we have to output data in a format like this:
{"x":[567,567,507,424,380,348,337,332,330,329,327,326,326],"y":[66,66,101,161,196,230,240,245,247,249,251,251,251],"t":[0,3,24,52,75,90,106,129,145,161,177,195,209]}

where x[i] is i-th x coorinate, y[i] - ith y coordinate, t[i] - i-th timestemp
